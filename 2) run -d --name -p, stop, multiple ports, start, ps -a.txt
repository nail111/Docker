----------------------------------------------------------------------------------------------------------------------------------

docker run -d [image]  --> detached mode, means that docker container runs in the background of your terminal.

Using detached mode also allows you to close the opened terminal session without stopping the container.

----------------------------------------------------------------------------------------------------------------------------------

docker run -d --name [container_name] -p 8080:8080 [image]

----------------------------------------------------------------------------------------------------------------------------------

docker stop [container_name]

----------------------------------------------------------------------------------------------------------------------------------

Multiple ports:

docker run -d -p 3000:80 -p 8080:80 [image]

----------------------------------------------------------------------------------------------------------------------------------

docker start [container_name]

----------------------------------------------------------------------------------------------------------------------------------

docker ps -a  --> shows all containers.

docker ps     --> shows all ran containers
