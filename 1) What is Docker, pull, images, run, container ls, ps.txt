----------------------------------------------------------------------------------------------------------------------------------

Docker  --> это инструмент, который позволяет запускать приложения в контейнере.

Image (Образ)  --> основной элемент из которого создаются контейнеры. Образ создаётся из Dockerfile, добавленного в проект.

Container  --> это абстракция на уровне приложения, объединяющая код и зависимости. Контейнеры всегда создаются из образов.

----------------------------------------------------------------------------------------------------------------------------------

docker pull  --> pull an image or a repository from a registry: docker pull nginx

----------------------------------------------------------------------------------------------------------------------------------

docker images  --> shows all images that downloaded.

----------------------------------------------------------------------------------------------------------------------------------

docker run  --> creates a container and starts it: docker run nginx:latest

----------------------------------------------------------------------------------------------------------------------------------

docker container ls  --> shows all run containers.

or

docker ps  --> same as docker container ls
